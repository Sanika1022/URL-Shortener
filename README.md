Title : URL-Shortener

Description : To convert large urls to short ones. The database of previously stortened urls will be stored ans we can search any of them in the Search Box.

How to run: After giving command "npm run devStart" you can open the required output on your browser by typing "localhost:5000".(make sure you have installed all the necessary - express, mongoose, ejs)

Theory : We have developed a 'URL Shortener' using node.js, express and MongoDB. HTML code is written in index.ejs file. package.jsom file is automatically created by giving the command "npm init".
